# NFT Mint Web Project

A modern NFT Minting Web App with wallet connectivity, motion animations, and BNB testnet deployment.

**Live Demo:** [View on Vercel](https://nft-mint-web-nu.vercel.app/)

---

## Features

- **Connect Wallet:** Connect any wallet (MetaMask, WalletConnect, etc.) seamlessly.  
- **Mint NFT:** Mint your own NFTs directly from the web interface.   
- **Motion Animations:** Smooth Framer Motion animations for a modern UX.  
- **BNB Smart Contract:** NFT contract deployed on the Binance Smart Chain (BNB testnet).  

---

## Tech Stack

- **Frontend:** React, Framer Motion  
- **Blockchain:** Solidity, ethers.js, BNB Testnet  
- **Storage:** IPFS for NFT metadata  
- **Deployment:** Vercel
